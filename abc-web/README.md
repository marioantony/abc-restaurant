### Required 
Node v16.20.2
npm 8.19.4

### Clone the project 
git@github.com:marioantony/abc-restaurant.git

### Front end instraction to start
1. open abc-web file
2. Give npm install
3. Start the project "npm start"

### Backend instruction to start
1. Open abc-backend
2. Give npm install
3. Start the project "node server.js"
